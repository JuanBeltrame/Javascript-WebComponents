Este es un modal básico creado con Web Components.

Puedes cerrarlo haciendo clic en el botón de cerrar, presionando ESC, o haciendo clic fuera del modal.

Características del Modal Component

  • Shadow DOM: Estilos encapsulados que no afectan al resto de la página
  • Accesibilidad: Soporte completo para teclado (ESC para cerrar, TAB trap)
  • Múltiples tamaños: small, medium, large
  • Eventos personalizados: open, close, confirm, cancel
  • Backdrop: Cierre al hacer clic fuera del modal
  • Animaciones: Transiciones suaves de entrada y salida
  • Responsive: Se adapta a diferentes tamaños de pantalla

Los Web Components nos permiten crear elementos reutilizables y encapsulados que pueden ser utilizados en cualquier proyecto sin conflictos de estilos o comportamiento.

Este modal component es un ejemplo perfecto de cómo crear componentes complejos con Vanilla JavaScript sin necesidad de frameworks externos.
